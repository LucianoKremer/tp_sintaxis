prerequisito tener mingw64 con y agregar a variables de entorno -> path -> el directorio de la carpeta bin (asi tenes los comandos como "gcc")

Como ejecutar:
	1- seguir el tutorial de este video de instalacion de flex y bison : https://youtu.be/yZ6Po_kEkD8?feature=shared
	2- Crear archivo objeto, abrir cmd o git bash en la carpeta y luego ejecutar comando "./Makefile" (esto deberia generar un par de archivos )
	3- para compilar el codigo de entrada : en la consola escribir "./main <codigoDeEntrada.txt "

importante --- reglas ---
	- no hay mayusculas
	- longitud maxima de identificador: 30
	- al crear una variable tiene que ser asignada en la misma linea que se crea
	- no esta definida la asignacion entre ids (ej mes_a es mes_n.)

el codigo a compilar es el .txt "entradaDeCodigo"
